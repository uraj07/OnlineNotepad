About Online Notepad :

Online Notepad is for editing plain text online, It is a free browser-based text editor that will help you to write documents in real-time on the internet. It is super simple tool which does not require any login or any user data. It is great for writing quick notes and can be used as a replacement of a paper or a notepad or a text file. Online Notepad provides basic useful features like copy, paste, undo, redo, font formatting, spell checker, word counter, character counter. It also has the option to download or save the drafted text. It is Designed and developed by Green Games. Feel free to contact d.old.green.games@gmail.com for any concern ! Browser Compatibility: Online Notepad supports modern web browsers : Google Chrome, Mozilla Firefox, Safari, Opera, Edge, Internet Explorer . You need to enable JavaScript in order to use the application. Uses of Online Notepad:
  • It makes writing notes easier and more convenient.
  • Due to its simple design, you will be able to concentrate more on your works.
  • It helps developers to work on the same browser window without switching to other Applications.
  • It helps you to write and edit plain text online, you do not need to install or launch other Software.
  • Students, Investors and Traders can quickly create online notes without installing text editing software.
  • It also provides inbuilt spell check which is very useful when you draft things very fast.
  • This tool can be used to quickly store things like holiday destinations, favorite places, and other important contents.
  • It is very helpful for the Bloggers, writers, and webmasters who can use it to note down their thoughts.
